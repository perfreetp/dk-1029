# 开放能力伙伴门户 - 产品需求文档

## 1. 产品概述

开放能力伙伴门户是一个面向外部合作方的一站式接入平台，提供平台账号、内容审核、消息推送和数据查询等核心能力的接入服务。平台通过标准化的流程管理，帮助合作伙伴快速完成能力接入、测试验证和上线运营，同时提供完善的账单管理和服务支持体系。

**目标用户**：外部企业合作伙伴、开发者、技术对接人员  
**核心价值**：降低接入门槛、提升对接效率、规范流程管理、透明化服务支持

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 企业管理员 | 企业入驻申请 | 全部功能权限、团队管理、账单管理 |
| 技术对接人 | 管理员邀请 | 工单管理、测试沙箱、技术对接 |
| 财务人员 | 管理员邀请 | 账单查看、续费管理 |

### 2.2 功能模块

1. **入驻申请页**：企业资料提交、联系人信息、回调地址配置
2. **资质审核页**：审核状态跟踪、资料补充、审核结果通知
3. **能力广场页**：能力浏览、权限边界查看、能力详情
4. **接入工单页**：工单发起、对接人分配、进度跟踪
5. **测试沙箱页**：测试参数下载、联调环境、结果提交
6. **上线验收页**：验收申请、验收标准、验收结果
7. **账单中心页**：配额查看、账单明细、续费停用
8. **服务评价页**：问题反馈、进度查询、服务评分

## 3. 核心流程

### 3.1 企业入驻流程

用户访问门户首页，点击"立即入驻"按钮进入入驻申请页面。填写企业基本信息（企业名称、统一社会信用代码、企业类型、行业分类），录入联系人信息（技术联系人、商务联系人、紧急联系人），配置回调地址，上传资质文件（营业执照、授权书等）。提交申请后进入资质审核阶段，平台审核人员对企业资料进行审核。审核通过后企业账号激活，可以开始接入能力；审核驳回则需根据驳回原因补充资料重新提交。

```mermaid
    "点击立即入驻" --> "填写企业信息"

    "录入联系人信息" --> "配置回调地址"
    "配置回调地址" --> "上传资质文件"
    "上传资质文件" --> "提交入驻申请"
    "提交入驻申请" --> "平台审核"

    "审核驳回" --> "补充资料"

    "重新提交" --> "平台审核"
    "审核通过" --> "账号激活"
    "账号激活" --> "开始接入能力"
```

### 3.2 能力接入流程

企业在能力广场浏览可接入的能力，查看能力详情和权限边界。选择目标能力后，在接入工单页面创建接入工单，填写接入需求和上传需求文档。平台分配对接人后，企业下载测试参数进入测试沙箱进行联调测试。联调完成后提交联调结果，申请上线验收。验收通过后能力正式上线，企业可以开始正式调用。

### 3.3 服务支持流程

企业在使用过程中遇到问题，可在服务评价页面提交问题反馈，选择问题类型、填写问题描述并上传截图。平台客服接收反馈后进行处理，企业可实时查询处理进度。问题解决后，企业对服务支持进行评分评价。

## 4. 用户界面设计

### 4.1 设计风格

**主题定位**：专业、可信、高效的企业级服务平台

**色彩方案**：
- 主色调：深蓝色系（#1E3A5F, #2C5282）- 传达专业、稳重、可信赖
- 辅助色：青蓝色（#38B2AC）- 用于强调和交互元素
- 成功色：翠绿色（#48BB78）- 表示成功、通过状态
- 警告色：琥珀色（#ED8936）- 表示警告、待处理状态
- 错误色：珊瑚红（#E53E3E）- 表示错误、驳回状态
- 背景色：浅灰蓝（#F7FAFC）- 页面背景
- 卡片背景：纯白色（#FFFFFF）

**字体方案**：
- 标题字体：思源黑体 / Noto Sans SC（粗体、中粗体）
- 正文字体：思源黑体 / Noto Sans SC（常规体）
- 代码字体：JetBrains Mono（用于技术参数展示）
- 字号层级：H1 32px、H2 24px、H3 20px、H4 16px、正文 14px、辅助 12px

**按钮样式**：
- 主要按钮：圆角 6px、高度 40px、填充主色、悬停加深
- 次要按钮：圆角 6px、高度 40px、边框主色、文字主色
- 文字按钮：无边框、文字主色、悬停下划线
- 危险按钮：填充错误色

**布局风格**：
- 顶部导航栏：固定定位、高度 64px、Logo + 导航菜单 + 用户信息
- 左侧菜单栏：宽度 240px、可折叠、图标+文字菜单项
- 内容区域：最小宽度 1200px、内边距 24px
- 卡片布局：圆角 8px、阴影 0 2px 8px rgba(0,0,0,0.08)
- 表格布局：斑马纹、悬停高亮、固定表头

**图标风格**：
- 使用线性图标（Line Icons）
- 统一线宽 2px
- 尺寸：小 16px、中 20px、大 24px

### 4.3 响应式设计

采用桌面优先设计策略，主要面向企业办公场景的大屏幕设备：

- **桌面端（≥1440px）**：完整布局，左侧菜单 + 右侧内容
- **中等桌面（1200px-1439px）**：紧凑布局，菜单可折叠
- **小屏桌面（992px-1199px）**：菜单折叠为图标模式
- **平板端（768px-991px）**：汉堡菜单，单列布局
- **移动端（<768px）**：底部导航，卡片堆叠

**触摸优化**：
- 按钮最小点击区域 44px × 44px
- 表格行高增大至 56px
- 滑动操作支持

### 4.4 动效设计

**页面过渡**：
- 路由切换：淡入淡出 200ms
- 卡片加载：骨架屏 + 渐显 300ms

**交互反馈**：
- 按钮悬停：背景色加深、轻微上移 2px
- 卡片悬停：阴影增强、轻微上浮 4px
- 表单聚焦：边框色变化、输入框轻微放大
- 加载状态：旋转加载图标 + 文字提示

**状态转换**：
- 审核进度：节点依次点亮动画
- 配额进度：进度条平滑填充
- 评分组件：星星依次点亮

**数据更新**：
- 列表刷新：新数据从顶部滑入
- 状态变更：颜色渐变过渡
- 数字变化：数字滚动动画
